# BookAI

Multi-agent AI system that generates complete books using structured pipeline architecture.

## Features
- Idea Generation Agent
- Outline Arranger Agent
- Chapter Writer Agent
- Copy Editor Agent
- Book Summarizer Agent

## Tech Stack
- FastAPI
- Python
- OpenAI API

## Run Project

Install dependencies:

pip install -r requirements.txt

Run server:

uvicorn app.main:app --reload
